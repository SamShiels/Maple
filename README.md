## Features
- Obj parser
- Custom materials
- Textures
- Scene hierarchy
- Cameras
- Directional lights
- Directional shadows
- Point lights
- Frustum culling
- Dynamic batching
- Post processing (single stack for now)
- Render textures (WIP)
